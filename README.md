# Patrimoine Cockpit

Application de gestion de patrimoine immobilier — HTML/CSS/JS pur.

## Structure
- `index.html` — point d'entrée
- `css/` — design-system.css + layout.css
- `js/` — data, ui-helpers, analytics, app, supabase

## Déploiement
Hébergé sur Netlify. Push sur `main` → déploiement automatique.

## Backend
Supabase (auth + sync cloud). Config saisie dans l'app (☁ Cloud → Configuration).
