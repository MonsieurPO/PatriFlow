Synchronisation cloud

Vos données sur tous vos appareils.

Configuration (2 minutes) :
1. Cliquez sur ☁ Cloud dans le header
2. Onglet Configuration → collez URL + clé Supabase
3. Onglet Connexion → créez votre compte
4. Synchronisation automatique activée ✓
💡 Vous pouvez commencer sans cloud et synchroniser plus tard.
